# 📋 AI Safety Triage Console

Demo MVP for safe, AI-assisted triage with human review, case management, and policy simulation.
https://mental-health-copilot.onrender.com/

## What it does

This app simulates an internal safety / triage console for reviewing user messages and routing them through a human-in-the-loop workflow.

Features:
- AI-assisted triage into LOW / MEDIUM / HIGH
- Priority mapping (P1 / P2 / P3)
- Human reviewer actions
- Case queue with status tracking
- Policy simulation (Conservative / Balanced / Lenient)
- Operational metrics such as backlog, escalation rate, and SLA-style overdue counts

## Important disclaimer

This is a demo prototype for portfolio and product demonstration purposes only.
It is **not** a substitute for professional medical or mental health care, and it is **not** intended for real-world crisis intervention.
